💡 HOW TO USE (STEP-BY-STEP):

Step 1: Prepare Files
  - Have all .nessus files in one folder
  - Or have a single .nessus file
  
Step 2: Run Main Generator
  $ python3 nessus-report-gen.py
  
Step 3: Answer Prompts
  1. "Enter .nessus file OR folder path:"
     → Enter path to your scan file(s)
  
  2. "Report Type: 1/2/3"
     → Enter: 3 (for Excel)
  
  3. "Output Mode: 1/2"
     → Enter: 1 (Combined into one file)
            or 2 (Separate files)
  
  4. "Level Filter: 1/2"
     → Enter: 1 (L1 + L2 all levels)
            or 2 (L1 critical only)
  
  5. "Output folder"
     → Press Enter (uses default: output/)
        or enter custom path
  
  6. "Output file name"
     → Press Enter (auto-generates with timestamp)
        or enter custom name
  
Step 4: Done!
  - Excel file(s) created in output folder
  - Ready to open and review
  - Share with stakeholders

🔧 TECHNICAL DETAILS:

New Function: build_excel_combined()
────────────────────────────────────
Purpose:
  Combines multiple .nessus file scans into one Excel worksheet

Parameters:
  • all_files_data: List of tuples
    - filename (str)
    - host_ip (Optional[str])
    - items (List[dict] of compliance data)
  
  • output_path: Where to save the .xlsx file

Logic:
  1. Create new Workbook
  2. For each file in all_files_data:
     a. Add green header row with file name
     b. Add blue header row with column names
     c. Add data rows with compliance items
     d. Add blank separator row
  3. Save file to output_path

Styling Applied:
  • Green (#70AD47): File headers
  • Blue (#B4C6E7): Column headers
  • Borders: All cells
  • Wrapping: Enabled
  • Alignment: Left/top
